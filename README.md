# Stillhet (Stillness)

> _"Silence is not about turning your back on your surroundings, but rather seeing the world more clearly."_ — Erling Kagge

**Stillhet** (Norwegian for _silence/stillness_) is a minimalist meditation web application inspired by Erling Kagge’s book _"Silence in the Age of Noise"_. The project is designed to provide users with a clean, distraction-free digital sanctuary to practice mindfulness, breathwork, and mental clarity amid the chaos of the modern world.

## 🛠️ Tech Stack

This project is built using a modern, scalable, and type-safe frontend stack:

- **Framework:** Vue 3 (Composition API)
- **Language:** TypeScript (for strict type safety and robust architecture)
- **State Management:** Pinia (modular and reactive global state)
- **Routing:** Vue Router (seamless Single Page Application navigation)
- **Build Tool:** Vite

## 🚀 Getting Started

To run this project locally, clone the repository and install the dependencies:

```bash
# Clone the repository
git clone https://github.com

# Navigate into the directory
cd stillhet

# Install dependencies
npm install

# Start the development server
npm run dev
```
